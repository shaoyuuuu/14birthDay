# Birthday Management System - Design System

## 设计理念

本设计系统遵循**简约、大气、和谐**的原则，提供一致的用户体验。

### 核心原则

1. **简约至上** - 去除不必要的装饰，聚焦内容
2. **视觉层次** - 通过颜色、大小、间距建立清晰的信息层次
3. **和谐统一** - 统一的设计语言和交互模式
4. **响应优先** - 适配各种设备和屏幕尺寸
5. **无障碍友好** - 确保所有用户都能轻松使用

## 使用指南

### 颜色使用
- **Primary**: 主要操作、CTA 按钮
- **Secondary**: 次要信息、辅助文本
- **Success**: 成功状态、确认操作
- **Warning**: 警告提示、需要注意
- **Danger**: 删除、取消等危险操作
- **Neutral**: 禁用、不可用状态

### 间距使用
- **紧凑**: 4-8px - 相关元素
- **标准**: 16-24px - 独立元素
- **宽松**: 32-48px - 主要区块

### 阴影使用
- **层级**: 根据元素重要性选择
- **一致性**: 相同层级使用相同阴影
- **克制**: 不要过度使用
