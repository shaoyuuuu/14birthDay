# 回忆内容管理

这个文件夹用于存放回忆内容，系统会自动读取并生成时间轴。

## 文件夹结构

```
memory/
├── 回忆1/
│   ├── memory.json      # 回忆描述文件
│   ├── image1.jpg       # 回忆图片1
│   └── image2.png       # 回忆图片2
└── 回忆2/
    ├── memory.json      # 回忆描述文件
    └── image1.jpg       # 回忆图片1
```

## memory.json格式

```json
{
  "date": "2023-01-15",
  "title": "回忆标题",
  "description": "回忆的详细描述",
  "memory": "回忆的内心感受或故事"
}
```

### 字段说明

- `date`: 回忆发生的日期（YYYY-MM-DD格式）
- `title`: 回忆的标题
- `description`: 回忆的详细描述
- `memory`: 回忆的内心感受或故事

## 使用方法

1. 在`memory`文件夹下创建一个新的文件夹（如"第一次旅行"）
2. 在该文件夹中创建`memory.json`文件，填写回忆内容
3. 将相关图片放入该文件夹中
4. 系统会自动读取并显示在时间轴上

## 注意事项

- 文件夹名称可以使用中文或英文
- 图片支持jpg、jpeg、png、gif、webp格式
- 系统会根据`date`字段自动排序
- 如果没有图片，`images`字段会自动为空
